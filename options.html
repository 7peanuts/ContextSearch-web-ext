— Import Method —

Install a tiny open-source app to import your search engine list and update automatically using Firefox's Native Messaging API when OpenSearch engines are added or removed. This app only runs when called by ContextSearch.

Import your installed search engines by browsing to your Firefox search engines data file. You typically need to re-import your list when you add OpenSearch engines.

— Import Search Engines —

 

Browse to your profile folder using the button above and select your search.json.mozlz4 file to import your One-Click Search Engines. Re-importing your search.json.mozlz4 file will keep your current settings and only add new engines. To completely replace your list on import, use the Remove All button.

You can find your profile path by opening about:profiles in the address bar. Typical locations:

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profile>/search.json.mozlz4

  Mac OS X

Users/<username>/Library/Application Support/Firefox/Profiles/<profile>/search.json.mozlz4

  Linux

/home/<username>/.mozilla/firefox/<profile>/search.json.mozlz4
