Load Search Engines
Firefox 57+ Add-ons cannot access your stored search engines automatically. Use the button below to load your search settings file.

Typical locations:

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profile>/search.json.mozlz4

  Mac OS X

Users/<username>/Library/Application Support/Firefox/Profiles/<profile>/search.json.mozlz4

  Linux

/home/<username>/.mozilla/firefox/<profile>/search.json.mozlz4


See the How-To video below for more details.

 

You will need to reload this file each time you change your search engine list
