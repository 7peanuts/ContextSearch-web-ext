Import Search Engines

Install a tiny open-source app to manage your search engine list automatically using Firefox's Native Messaging API. This app only runs when called by ContextSearch.

Import your search engine list by browsing to your Firefox search engines file. You will need to manually import your list when you add or remove search engines.

Automatic
Step 1: Install app
Windows installer  [more info]
source
source
Step 2: Enter the full path to your profile
You can find your profile path by opening about:profiles in the address bar
Manual
Browse to your profile folder using the button below and select your search.json.mozlz4 file
You can find your profile path by opening about:profiles in the address bar

 

Typical locations:

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profile>/search.json.mozlz4

  Mac OS X

Users/<username>/Library/Application Support/Firefox/Profiles/<profile>/search.json.mozlz4

  Linux

/home/<username>/.mozilla/firefox/<profile>/search.json.mozlz4
