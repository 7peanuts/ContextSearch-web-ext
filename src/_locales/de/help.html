Einstellungen

Die meisten Einstellungen können unter dem Reiter Menüs angepasst werden. Fahren Sie mit der Maus über ? Icons, um Details zu jeder Einstellung zu sehen.

Verwendung des Kontextmenüs

Markieren Sie ein Wort und klicken Sie mit der rechten Maustaste, um das Kontextmenü zu öffnen. Erweitern Sie das Kontextmenü mit der Bezeichnung Suche nach "...." und wählen Sie, mit welcher Suchmaschine Sie suchen möchten.

Das Aktivieren von Lesezeichenmenü verwenden erzeugt den Ordner ContextSearch Menu unter Weitere Lesezeichen wo Sie Suchmaschinen neu ordnen, sowie Unterordner und Trennlinien erstellen können. Das Umbenennen von Suchmaschinen-Lesezeichen führt dazu, dass diese nicht mehr funktionieren. Gelöschte Lesezeichen können aus dem Suchmaschinen-Manager unter dem Reiter Suchmaschinen wiederhergestellt werden.

Verwendung des Schnellmenüs

Standard: Markieren Sie ein Wort und halten Sie die rechte Maustaste gedrückt, bis das Schnellmenü erscheint. Sie können im Reiter Menüs unter Öffnen & Schließen aus mehreren Optionen wählen, wie sich das Schnellmenü öffnet.

Das Schnellmenü kann über den Reiter Menüs angepasst werden. Suchmaschinen können im Suchlistenmanager über den Reiter Suchmaschinen neu angeordnet werden.

Firefox-Suchmaschinen importieren

Wenn Sie der Firefox-Suchsymbolleiste Suchmaschinen hinzugefügt haben, die Sie mit ContextSearch verwenden möchten, können Sie diese importieren. Wählen Sie dafür die Schaltfläche 'Ein-Klick-Suchmaschinen importieren' im Reiter Suchmaschinen, gehen Sie dann zu Ihrem Profilordner und wählen dort die Datei search.json.mozlz4 aus. Sie finden Ihren Profilpfad, indem Sie about:profiles in der Adressleiste öffnen. Typische Speicherorte:

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profile>/search.json.mozlz4

  Mac OS X

Users/<username>/Library/Application Support/Firefox/Profiles/<profile>/search.json.mozlz4

  Linux

/home/<username>/.mozilla/firefox/<profile>/search.json.mozlz4

Automatischer Import

Optional können Sie eine kleine Open-Source-Anwendung installieren, um Ihre Suchmaschinenliste automatisch zu importieren. Dies geschieht dann über Firefox's Native Messaging API, sobald OpenSearch Engines hinzugefügt oder entfernt werden. Diese App läuft nur, wenn sie von ContextSearch aufgerufen wird. Nützlich beim Hinzufügen von OpenSearch-Engines aus anderen Quellen als ContextSearch, wofür normalerweise ein Import erforderlich wäre.

Hinzufügen von Suchmaschinen

Verwenden des Menüs Benutzerdefinierte Suche hinzufügen

Wenn eine Website ein Suchformular anbietet, klicken Sie einfach mit der rechten Maustaste auf das Formular, in das Sie Ihre Suchbegriffe eingeben würden, und wählen Sie den Menüpunkt Benutzerdefinierte Suche erstellen aus dem Kontextmenü. Es erscheint ein Popup, in dem Sie aufgefordert werden, der neuen Suchmaschine einen Namen zu geben. Der Name muss eindeutig sein. Klicken Sie auf 'Hinzufügen' und Ihre neue Suchmaschine wird zu ContextSearch hinzugefügt. Als nächstes werden Sie gefragt, ob Sie Ihre neue Suchmaschine zur Firefox-Suchleiste hinzufügen möchten. Klicken Sie auf Ja, um eine Anfrage für eine neue OpenSearch-Engine zu senden. Bestätigen Sie die Installation und Ihre neue Suchmaschine wird auch in der Firefox-Suchleiste angezeigt. Diese Engines bleiben installiert und nutzbar, auch wenn Sie ContextSearch deinstallieren.

Benutzerdefinierte Suche erstellen bietet auch erweiterte Einstellungen mit mehr Optionen zum Hinzufügen einer neuen Suchmaschine.

Verwendung des Suchmaschinenmanagers

Installierte Suchmaschinen können über den Manager im Reiter Suchmaschinen geändert werden.

Verbergen

Klicken Sie auf das Kontrollkästchen, um Suchmaschinen in Ihrer Kontextmenü- oder Schnellmenü-Liste ein- oder auszublenden. Wenn Lesezeichenmenü verwenden aktiviert ist, wirkt sich das Ausblenden nur auf das Schnellmenü aus.

Sortieren

Durch Ziehen & Loslassen können Sie die Suchmaschinen in der Liste neu ordnen. Wenn Lesezeichenmenü verwenden aktiviert ist, wirkt sich das Ausblenden nur auf das Schnellmenü aus.

Löschen

Klicken Sie auf das Symbol , um zum Löschen einer Suchmaschine aufgefordert zu werden. OpenSearch-Suchmaschinen, die in der Firefox-Suchleiste erscheinen, sind nicht betroffen und müssen separat von den Firefox-Einstellungen->Suche->Ein-Klick-Suchmaschinen entfernt werden. Wenn Automatischer Import aktiviert ist, werden gelöschte Suchmaschinen, die als Ein-Klick-Suchmaschinen existieren, wiederhergestellt. Sie können diese Suchmaschinen einfach ausblenden, wenn sie wieder auftauchen.

Bearbeiten

Wenn Sie auf den Namen einer Suchmaschine klicken, wird ein Formular geöffnet, in dem Sie Suchmaschinenparameter wie Abfragevorlage, Kodierung, Symbol usw. bearbeiten können.

Lesezeichen

Wenn Lesezeichenmenü verwenden aktiviert ist, erscheint für jede Suchmaschine das . Hiermit können Sie Suchmaschinen, die im Lesezeichen-Ordner erscheinen, für die Verwendung mit dem Kontextmenü umschalten. Aktivierte Suchmaschinen werden an das Ende der Lesezeichenliste im ContextSearch Menu angehängt. Sie können Suchmaschinen aus dem Ordner ContextSearch Menu unter Lesezeichen verwalten neu ordnen.
