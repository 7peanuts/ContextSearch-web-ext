Settings

Most settings can be customized under the Menus tab. Hover your mouse over ? icons to see details about each setting.

Using the Context Menu

Highlight a word and right-click to open the context menu. Expand the ContextSearch menu labeled Search for "..." and choose which search engine to perform the search with.

Enabling Use Bookmarks Menu creates the folder ContextSearch Menu under Other Bookmarks where you can reorder search engines, create subfolders and separators. Renaming search engine bookmarks will cause them to no longer work. Deleted bookmarks can be restored from the search engines manager under the Search Engines tab.

Using the Quick Menu

Default: Highlight a word and hold down the right mouse button until the Quick Menu appears. You can choose from several options how the quick menu opens in the Menus tab under Opening & Closing

The Quick Menu can be customize from the Menus tab. Search engines can be rearranged in the search list manager from the Search Engines tab.

Importing Firefox Search Engines

If you have added search engines to the Firefox search toolbar you would like to use with ContextSearch, you can import them by clicking the button under 'Import One-Click Search Engines' in the Search Engines tab. Browse to your profile folder and select the search.json.mozlz4 file. You can find your profile path by opening about:profiles in the address bar. Typical locations:

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profile>/search.json.mozlz4

  Mac OS X

Users/<username>/Library/Application Support/Firefox/Profiles/<profile>/search.json.mozlz4

  Linux

/home/<username>/.mozilla/firefox/<profile>/search.json.mozlz4

Adding Search Engines

Using the Add Custom Search menu

If a website provides a search form, simply right-click the form where you would enter your search terms and select the menu item Add Custom Search from the context menu. A popup will appear asking you give the new search engine a name. The name must be unique. Click 'Add' and your new search engine will be added to ContextSearch. Next you will be asked if you want to add your new search engine to the Firefox search toolbar. Clicking yes will send a request for a new OpenSearch engine. Confirm the install and your new engine will also appear in the Firefox search toolbar. These engines will remain installed and usable even if you uninstall ContextSearch.

Add Custom Search also provides advanced settings with more options for adding a new search engine
Add
Install a new search engine based on the official search engine offered by the website
Find
Search Mycroftproject for search engines related to the current website made by users around the world. You will need to reimport your One-Click Search Engines to use engines installed with this method in ContextSearch
Create
Full control over creating a new search engine
