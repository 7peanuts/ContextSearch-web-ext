Configuration

La plupart des réglages sont configurables à l’onglet Menus. Passez votre souris au-dessus des icônes ? pour avoir des informations concernant chaque réglage.

Utilisation du menu contextuel

Sélectionnez un mot et faites un clic droit pour ouvrir le menu contextuel. Développez le sous-menu Rechercher « mot » avec… et choisissez le moteur de recherche avec lequel vous souhaitez effectuer la recherche.

L’activation de l’option Organiser avec les marque-pages crée le dossier Recherche contextuelle dans Autres marque-pages. Dans ce dossier, vous pouvez réorganiser les moteurs de recherche, créer des sous-dossiers et ajouter des séparateurs. Si vous les renommez, ils ne fonctionneront plus. Si vous les supprimez, ils pourront être rétablis avec le gestionnaire des moteurs de recherche à l’onglet Moteurs de recherche.

Utilisation du menu rapide

Sélectionnez un mot et maintenez le bouton droit de la souris enfoncé jusqu’à ce que le menu apparaisse. Ce mode d’ouverture par défaut peut être modifié dans Ouverture et fermeture à l’onglet Menus.

Le menu rapide est configurable à l’onglet Menus. Les moteurs de recherche peuvent être réorganisés dans la liste de l’onglet Moteurs de recherche.

Importation des moteurs de recherche de Firefox

Si vous avez ajouté des moteurs de recherche dans la barre de recherche de Firefox et que vous souhaitez les utiliser avec ContextSearch, il vous suffit de les importer en sélectionnant le fichier search.json.mozlz4 dans votre profil Firefox. Pour cela, cliquez sur le bouton Parcourir… sous Importation des moteurs de recherche accessibles en un clic à l’onglet Moteurs de recherche. Vous trouverez le chemin de votre profil en ouvrant about:profiles dans la barre d’adresse. Emplacements habituels :

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profil>/search.json.mozlz4

  Mac OS X

Users/<utilisateur>/Library/Application Support/Firefox/Profiles/<profil>/search.json.mozlz4

  Linux

/home/<utilisateur>/.mozilla/firefox/<profil>/search.json.mozlz4

Ajout d’un moteur de recherche

Utilisation du dialogue d’ajout

Si un site web dispose d’un formulaire de recherche, faites un clic droit là où vous devez normalement saisir les termes de la recherche et sélectionnez l’élément de menu contextuel Ajouter ce moteur de recherche. Apparaît alors une fenêtre vous demandant de donner un nom à ce nouveau moteur. Ce nom doit être unique. Cliquez sur Ajouter pour ajouter le moteur à ContextSearch. Il vous est ensuite demandé si vous souhaitez l’ajouter à la barre de recherche de Firefox. Cliquez sur Oui pour envoyer la requête d’ajout, confirmez l’installation pour le voir apparaître dans la barre de recherche. Ce moteur restera installé et fonctionnel, même si vous désinstallez ContextSearch.

Le dialogue d’ajout d’un moteur de recherche dispose également d’un mode avancé proposant davantage d’options :
Ajouter
Installe le moteur de recherche officiel fourni par le site web.
Trouver
Recherche un moteur de recherche compatible avec le site chez Mycroftproject. Vous aurez à réimporter vos moteurs de recherche accessibles en un clic pour pouvoir utiliser dans ContextSearch un moteur installé de cette façon.
Créer
Vous contrôlez intégralement la création du moteur de recherche.

Gestion des moteurs de recherche

Les moteurs de recherche installés peuvent être modifiés dans la liste à l’onglet Moteurs de recherche.

Masquage

Cliquez sur la case à cocher pour faire apparaître ou non un moteur de recherche dans le menu contextuel ou dans le menu rapide. Si l’option Organiser avec les marque-pages est cochée, seul le menu rapide est concerné.

Réorganisation

Procédez par glisser-déposer pour réorganiser votre liste. Si l’option Organiser avec les marque-pages est cochée, seul le menu rapide est affecté par cette réorganisation.

Suppression

Cliquez sur l’icône pour supprimer un moteur de recherche. Les moteurs OpenSearch figurant dans la barre de recherche de Firefox ne peuvent pas être supprimés de cette façon, il faut passer par les options de Firefox (Recherche -> Moteurs de recherche accessibles en un clic). Avec l’importation automatique activée, les moteurs de recherche accessibles en un clic existants réapparaissent, il suffit alors de les masquer.

Édition

Cliquez sur le nom d’un moteur de recherche pour ouvrir un formulaire qui vous permettra de modifier ses paramètres comme le modèle de requête, l’encodage, l’icône, etc.

Marquage

En cochant l’option Organiser avec les marque-pages, chaque moteur de recherche se voit doté d’une nouvelle icône permettant de le faire apparaître ou non dans le menu contextuel. Un moteur marqué est ajouté en fin de liste du dossier Recherche contextuelle situé dans les marque-pages. Dans ce dossier, vous avez la possibilté d’organiser les moteurs de recherche comme bon vous semble.
