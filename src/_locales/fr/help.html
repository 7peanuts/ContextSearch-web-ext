Configuration

La plupart des réglages se font au niveau des différents onglets. Passez votre souris au-dessus des icônes ? pour avoir des informations concernant certaines options.

Utilisation du menu contextuel

Sélectionnez un mot et faites un clic droit pour ouvrir le menu contextuel. Développez le sous-menu Rechercher « mot » avec et choisissez le moteur de recherche avec lequel vous souhaitez effectuer la recherche.

Utilisation du menu instantané

Sélectionnez un mot et maintenez le bouton droit de la souris enfoncé jusqu’à ce que le menu apparaisse. Ce mode d’ouverture par défaut peut être modifié à la rubrique OUVERTURE de l’onglet Menu instantané.

Importation des moteurs de recherche de Firefox

Si vous avez ajouté des moteurs de recherche dans la barre de recherche de Firefox et que vous souhaitez les utiliser en recherche contextuelle, il vous suffit de les importer en sélectionnant le fichier search.json.mozlz4 dans votre profil Firefox. Pour cela, cliquez sur le bouton Parcourir… sous Importation des moteurs de recherche accessibles en un clic à l’onglet Moteurs de recherche. Vous trouverez le chemin de votre profil en ouvrant about:profiles dans la barre d’adresse. Emplacements habituels :

  Windows

%APPDATA%/Mozilla/Firefox/Profiles/<profil>/search.json.mozlz4

  Mac OS X

Users/<utilisateur>/Library/Application Support/Firefox/Profiles/<profil>/search.json.mozlz4

  Linux

/home/<utilisateur>/.mozilla/firefox/<profil>/search.json.mozlz4

Ajout d’un moteur de recherche

Utilisation du dialogue d’ajout

Si un site web dispose d’un formulaire de recherche, faites un clic droit là où vous devez normalement saisir les termes de la recherche et sélectionnez l’élément de menu contextuel Ajouter ce moteur de recherche. Apparaît alors une fenêtre vous demandant de donner un nom à ce nouveau moteur. Ce nom doit être unique. Cliquez sur Ajouter pour ajouter le moteur à la recherche contextuelle. Il vous est ensuite demandé si vous souhaitez l’ajouter à la barre de recherche de Firefox. Cliquez sur Oui pour envoyer la requête d’ajout, confirmez l’installation pour le voir apparaître dans la barre de recherche. Ce moteur restera installé et fonctionnel, même si vous désinstallez l’extension.

Le dialogue d’ajout d’un moteur de recherche dispose également d’un mode avancé proposant davantage d’options :
Ajouter
Installe le moteur de recherche officiel fourni par le site web.
Trouver
Recherche un moteur de recherche compatible avec le site chez Mycroftproject. Vous aurez à réimporter vos moteurs de recherche accessibles en un clic pour pouvoir utiliser en recherche contextuelle un moteur installé de cette façon.
Créer
Vous contrôlez intégralement la création du moteur de recherche.
