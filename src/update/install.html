Import Search Engines

Since Firefox 57, all add-ons use the WebExtensions standard. This limits what information an add-on can access about your browser. Features like Firefox's One-Click search engines can't be accessed directly. ContextSearch web-ext gets around this limitation in two ways

Automatic

The WebExtensions Native Messaging API includes a way for Firefox add-ons to talk to 3rd-party applications. Install a tiny open-source app to allow ContextSearch web-ext to access your search engines file automatically whenever you install or remove new search engines.

Manual

ContextSearch can access One-Click Search Engines by having the user browse directly to the file the engines are stored in. Using the manual method will require you to repeat this process when you add or remove search engines to keep ContextSearch updated with the lasted information. If you rarely add new search engines and really don't want to install the native app, this method is for you.

Modify Search Engines

To see your currently installed search engines, go to Options->Search->One-Click Search Engines. Here you can remove, reorder and hide Firefox's OpenSearch plugins.

Add Search Engines

Most websites have a custom search bar near the top of page. Right-click the text input field to bring up the context menu. You should see the option 'Add Custom Search'. Click to show the Add Custom Search dialog


Add

If the website has an OpenSearch plugin available to download, the Add button will appear. Click to install.

Find

Search MycroftProject.com for plugins for the current website.

Create

Make a custom search engine for the current input field. If the input is part of a <form>, values will be set automatically. Some forms will require custom settings.

* Custom engines will be served by opensearch-api.appspot.com. Your engine request may be denied if this server reaches its bandwidth limit.

* If you are using ContextSearch-lite or using the Manual method of updating your search engine list, you will need to reload your search.json.mozlz4 file for the changes take effect. If using the Automatic setting with the ContextSearch native app, your search engine list will update automatically.
Please refresh your search engines in Options to enable features
