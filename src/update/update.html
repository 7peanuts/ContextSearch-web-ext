* Attention ContextSearch web-ext LITE Users *

ContextSearch web-ext Lite has been depreciated, no longer receiving updates and will be removed from the add-on store. ContextSearch web-ext Lite users should export their settings and switch to ContextSearch web-ext 1.8.0+ to continue receiving updates for ContextSearch web-ext.

Native Messaging permissions have been moved from the standard version to a separate add-on, so there are no permissions differences between the standard v1.8.0 and the previous lite versions.

Changes for 1.8.0

* a lot of changes were made for this update, so if you have any issues report them here and I'll get a fix up ASAP

    Search Engine Manager

  • All menus (context, quick, toolbar) use the same search engine list. If you are using a mix of Manage Using Bookmarks and the Search Engine Manager for separate Context and QuickMenu layouts, you may need to edit your list.
  • Search Engine Manager handles subfolders, separators and bookmarklets eliminating the need for Bookmarks
  • Search Engine Manager actions are handled by a custom context menu (right-click)
  • Bookmarklets can be added from the manager via context menu. This menu will be populated by bookmarklets found anywhere in Bookmarks
  • Search Regex option added to Edit Search Engine form in manager. Current format is "regex to match", "string to replace". For example, " ", "+" will replace spaces with + in the search terms.

    • Menus

  • Added option to use a custom shotcut key to open the search engines subfolder from the context menu (highlight word, right click, press shortcut key)
  • QuickMenu setting Items has been superseded by Rows. Columns and Rows set the initial dimensions of the Quick Menu
  • QuickMenu includes More button to expand from the initial size to display ALL items.
  • If QuickMenu Columns and Toolbar Columns are equal, Toolbar layout will match QuickMenu layout, accounting for Tools.
  • QuickMenu can be resized while open by dragging lower-right corner

    • Firefox Users and Auto Import

  • Native App support has been moved to a separate addon to simplify cross-browser support. Automatic Import users should download the ContextSearch Auto Importer addon from mozilla and the updated native app from github.

    • Fixes

  • Fixed {ENTER} not performing search
