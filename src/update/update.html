Changes for 1.8.0

Search Engine Manager

Search Engine Manager handles subfolders, separators and bookmarklets eliminating the need for Bookmarks

All menus (context, quick, toolbar) use the same search engine list. If you are using a mix of Manage Using Bookmarks and the Search Engine Manager for separate Context and QuickMenu layouts, you may need to edit your list.

Native App support has been moved to a separate addon to simplify cross-browser support. Automatic Import users should download the ContextSearch web-ext Native Messenger addon package from the repository.

ContextSearch Lite has been depreciated and will be removed from Addons. ContextSearch Lite users should export their settings and switch to ContextSearch

QuickMenu setting Items has been superseded by Rows. Columns and Rows determine the initial size of the Quick Menu QuickMenu includes More button to expand from the initial size to display ALL items.

If QuickMenu Columns and Toolbar Columns are equal, Toolbar layout will match QuickMenu layout, accounting for Tools.
